# E-Commerce-Backend

## Description

Tasked to build e-commerce back end that uses Express and Sequelize to interact with a MySQL database. By calling to the API, users can retrieve data based on category, product name, and product tag (i.e. 'gold', 'rock music').

[Video walkthrough](https://drive.google.com/file/d/1rMIFzUmtUtUWZLoSnAPE_J2IDyBKwytG/view?usp=sharing)

## Installation & Usage

After installing dependencies, create your `.env` file and connect to the mySQL database. Run the seeds file to initialize some content into the database. Start up the server and then use a tool such as Insomnia or Postman to view the responses for each route.
